>

## Install

```bash
npm install --save @wassimbenzarti/react-components
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from '@wassimbenzarti/react-components'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
